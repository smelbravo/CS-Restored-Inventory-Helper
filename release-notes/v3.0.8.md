### v3.0.8

Fixes page freezing / "This page isn't responding" on large inventories (Brave, Edge, Chrome), especially with 1000+ items.

- Float/seed overlays update incrementally — no longer tear down and rebuild every 2 seconds
- Browse filters no longer re-run on a background poll loop
- Background refresh every 8s (uses idle time when the browser is free)
- Faster first load: uses cached API data, quicker overlay refresh, browse bar retries while the SPA loads
- **Fix:** inventory float/seed missing (`fetchInventory` did not update `inventoryCache`)
- **Fix:** inventory API responses no longer misclassified as marketplace data
- Name-based item matching on inventory when image ID matching fails

### Install

1. Download `CS-Restored-Inventory-Helper-v3.0.8.zip` below
2. Unzip
3. **Firefox:** `about:debugging` → This Firefox → Load Temporary Add-on → `manifest.json`
4. **Chrome / Edge / Brave:** Extensions → Developer mode → Load unpacked → select the unzipped folder
