## v3.4.0 — Auto case opening, lazy overlays, UI contrast

### Highlights

- **Auto case opening** — new **Auto open** tab on [Cases](https://csrestored.fun/app/inventory/cases): set delay, time limit, and coin spend limit; live log while opening; **Results** table sorted by float (best first) when done; **Stop** anytime (toggle in extension popup, default off).
- **Lazy overlay loading** — on grids with **80+** item cards, float/seed badges load on visible cards first and more appear as you scroll (inventory, marketplace, trades, trade modals). Helps reduce lag with **200+** skins.
- **Quick Sell panel** — brighter text on section labels, status, dropdown, and batch-size label for easier reading.

### Install — Firefox (recommended)

**[Install from Firefox Add-ons](https://addons.mozilla.org/firefox/addon/csr-inventory-helper/)** — signed build after AMO approves **v3.4.0**; until then use the `.zip` below.

### Install — Chrome, Edge, Brave

1. Download **`CS-Restored-Inventory-Helper-v3.4.0.zip`** below
2. Unzip the folder
3. Open **Extensions** → enable **Developer mode**
4. **Load unpacked** → select the unzipped folder (the one that contains `manifest.json`)

### Updating

- **Firefox:** updates automatically from [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/csr-inventory-helper/) after AMO approval
- **Chrome / Edge / Brave:** replace the folder and click **Reload**, or remove and load again
